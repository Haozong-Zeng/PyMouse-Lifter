# PyMouse-Lifter
This repository contains the codes and documents of the "PyMouse-Lifter: Real Time 3-D Pose Estimation for Mice with Only 2-D Annotation Via Data Synthesis" project in Dr. Tim Murphy's Lab at UBC.<br />
Blender project: to run the Python script in Blender, please install the dependencies in Blender's own Python environment.<br />
